# SSAccount_Spider
这是一个mac版shadowsocks账号爬虫工具，爬取ss.ishadowx.net的免费ss账号并导入到shadowsocks客户端<br>

## SS App
download ShadowsocksX:
https://us.ishadowx.net/

## Usage
依赖包：bs4，安装：`pip install bs4`

1. `python Spider.js`

generate ss_cfg.json, shadowsocks.xml and shadowsocks.plist

2. ``./run.sh`
